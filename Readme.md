# dotfiles

My nixos setup for linux & macOS

## Directives

**Machines** (NixOS only)

Holds the hardware configuration of a physical device.
- Boot configuration
- Hardware specific stuff

**System** (NixOS only)

Has all system related configuraion needed for NixOS which are not part of User or Machine.
- System packages
- Virtualisation
- User configuration (e.g: groups)

**Home**

Holds all home-manager related configuration.
- Desktop config
- shell config
- User packages
- User software
- dotfiles
